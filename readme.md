Slippers
========

A self hosted API testing tool
------------------------------

Slippers helps you develop your own api by documenting resources and available parameters.

When tools like apigee are overkill or simply not an option, Slippers can help.

**Features**

* Scripted API resources
* Basic Authentication
* Syntax Highlighting (Geshi)

**Requirements**

* PHP
* CURL

**Roadmap**

* Provide example APIs
* Improvements on resource URL customization
* Generate API documentation
* OAuth
* Customize parameters input type
